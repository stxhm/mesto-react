# Проект 10: Место
---

Проект представляет собой адаптивный одностраничный сайт-приложение с возможностью редактировать профиль пользователя, добавлять и удалять карточки с фотографиями. Использованы следующие технологии:

* HTML
* CSS (flexbox, grid)
* BEM, Nested BEM
* React и JSX
* Webpack
